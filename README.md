# Ювелирный салон "Бриллиант"

Консольное приложение для управления ювелирным магазином.

## Функциональность
- Просмотр каталога украшений
- Поиск и фильтрация (по металлу, пробе, камням, цене)
- Регистрация клиентов и система лояльности
- Оформление заказов (подарочная упаковка, гравировка, рассрочка)
- Личный кабинет с историей и рекомендациями
- Статистика продаж

## Технологии
- C#
- .NET 9.0
- Git / GitHub
